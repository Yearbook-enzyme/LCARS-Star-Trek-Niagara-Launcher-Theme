Setup path

  1. Download the Niagara theme file. Import the included .nlt file into Niagara Launcher.
  2. Generate a matching LCARS wallpaper. Pick your device size, palette, spine position, thickness, segment length, and panel rhythm.
  3. Install Trek-inspired fonts locally. Open the Font Guide, install fonts from their original sources, then preview them against a Niagara-style app list.
  4. Export your Niagara app list. Use Niagara's export so the icon generator can categorize your actual apps.
  5. Build the LCARS icon pack. Review categories, fix anything weird, and build the signed APK.
  6. Apply and iterate. Use the theme, wallpaper, font choice, and icon pack together; share reviewed mappings to improve future matching.
