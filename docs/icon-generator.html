Color mapping controls how the selected palette gets distributed across app categories. Use Palette sweep or Flag stripes for rainbow, pride, and regional flag palettes.

Niagara folder note: generated APKs include LCARS folder drawables for launchers and manual icon pickers that expose folder-icon selection. Niagara Launcher appears to keep its folder/pop-up icons internal, so those folder icons may not auto-apply in Niagara.
Before sharing or building: parse your app list and review the categories in the table. Correcting categories improves LCARS color grouping and saves your choices locally for next time. To improve the shared database, use both sharing controls: check Share parsed app mappings, then click Submit Reviewed Mappings. If you build a signed APK while sharing is checked, the builder will also submit the reviewed mappings.
