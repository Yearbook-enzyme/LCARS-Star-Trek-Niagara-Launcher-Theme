How to Use the LCARS Icon Generator

The icon generator is privacy-first. It does not read your device, scan your apps, or upload your app list. You provide only the list you want to use.

Recommended method

Simple manual list

Type or paste one app name per line. This works on Android, iPhone, iPad, desktop, and any other device. 

Discord
Brave
ChatGPT
Photos

This is the safest and easiest method. The generator will guess categories, and you can edit them before downloading icons.

Android

App names or package names

Android icon packs work best with package names, but you do not need them for basic generated icons.

App names are fine:

Discord
Brave
ChatGPT

Package names are more precise if you already have them:

com.discord
com.brave.browser
com.openai.chatgpt

If you already know how to export package names from your launcher, backup tool, desktop tools, or another trusted method, you can upload that list. This site does not require you to install anything or connect your phone to a computer.

iPhone and iPad

Paste app names manually, generate PNG icons, then use Apple Shortcuts to add custom icons to your Home Screen.

On iPhone and iPad, generated icons are meant for Shortcuts/Home Screen setup, not Android-style icon packs.

You can mix app names and package names. The generator will deduplicate entries and let you correct categories before export.

Privacy note

Your app list can reveal personal habits, so the generator runs locally in your browser. Unknown app contribution JSON is copied only when you choose to copy it.
