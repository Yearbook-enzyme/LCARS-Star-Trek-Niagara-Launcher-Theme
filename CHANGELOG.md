# Changelog

## v0.2.0 — LCARS workflow, wallpaper, icon, and typography update

### Added
- Added guarded wallpaper geometry so extreme slider settings are less likely to break the LCARS elbow layout.
- Added **Panel rhythm** options for more expressive static wallpaper compositions.
- Added a Font Guide page with Trek-inspired font recommendations.
- Added **Federation** as the recommended readable default font direction for this theme.
- Added font selection as a visible step in the Start Here flow.
- Added reviewed mapping submission flow for improving shared app-category matching.
- Added import reporting support for app-category contribution imports.

### Improved
- Updated app-category data from reviewed contributed mappings.
- Improved the category-review workflow before building the signed icon-pack APK.
- Clarified that Star Trek-inspired fonts should be previewed/downloaded from their original source rather than redistributed by this project.
- Improved the main setup path so wallpaper, fonts, app list export, icons, APK build, and final Niagara setup are presented as one flow.

### Notes
- This release does not bundle Star Trek font files.
- Font installation and testing should happen on the user's own device or launcher setup.
- The Niagara theme file itself is still static; the generated wallpaper and icon-pack workflow are the main customizable parts.
